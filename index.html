⚙ Conectar ao Excel do OneDrive

Configure uma vez e pronto — sempre que atualizar seu Excel, clique em Sincronizar para o painel refletir as mudanças automaticamente.

Obtido em: portal.azure.com → Azure Active Directory → App registrations → seu app → Application (client) ID
Não tem o app ainda? Veja o guia de configuração.
Caminho relativo à raiz do seu OneDrive. Exemplo: se o arquivo está em Meus Documentos → RH → arquivo.xlsx, escreva Meus Documentos/RH/arquivo.xlsx
Nome exato da aba do Excel que contém os dados (padrão: GERAL)
📋 Como criar o app no Azure (passo a passo)
1. Criar o appAcesse portal.azure.com → Azure Active Directory → App registrations → New registration. Nome: "Painel Benefícios". Tipo: "Accounts in this organizational directory only". Clique em Register.
2. Copiar o Client IDNa tela do app recém-criado, copie o valor de "Application (client) ID" e cole no campo acima.
3. Configurar redirect URIVá em Authentication → Add a platform → Single-page application. Redirect URI: cole aqui o endereço exato da URL do seu painel HTML no SharePoint/OneDrive.
4. Adicionar permissãoVá em API permissions → Add a permission → Microsoft Graph → Delegated permissions → busque "Files.Read" → marque → Add permissions.
5. ProntoSalve as configurações aqui e clique em Sincronizar. Na primeira vez será pedido login com sua conta Microsoft.
